# transtypro phase skill

When this skill is invoked:

1. Read:
   - SOUL.md
   - AGENTS.md
   - docs/PHASES.md
   - docs/TASK_BOARD.md
   - docs/PROGRESS.md

2. Identify the phase.

3. Produce:
   - phase goal
   - files likely to change
   - implementation steps
   - test plan
   - privacy impact
   - rollback plan

4. Implement only the approved phase scope.

5. After implementation:
   - run checks
   - update docs/PROGRESS.md
   - update docs/TASK_BOARD.md
   - create or update handoff notes
