# transtypro feature skill

For one feature only:

1. Define the feature.
2. Confirm acceptance criteria.
3. Identify frontend files.
4. Identify backend files.
5. Identify database changes.
6. Identify privacy impact.
7. Implement the smallest working version.
8. Connect UI to real backend commands.
9. Add tests where practical.
10. Update docs.

Do not mix unrelated features.
